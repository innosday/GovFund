# Project GovFund: Strategic Funding Command Center

## 1. Project Overview
* **Goal:** 단순한 공고 조회를 넘어, 기업의 데이터를 기반으로 **'당첨 확률'을 극대화**하고 **'서류 작성을 자동화'**하는 기업 전용 자금 조달 사령부.
* **Target User:** R&D 자금이 절실하지만 행정 역량이 부족한 기술 중심 스타트업 및 중소기업.
* **Core Value:** 
    * **Intelligence:** 기업 데이터(매출, 특허, 인원) 분석을 통한 정밀 매칭 및 합격 확률 예측.
    * **Automation:** Vault(금고) 데이터와 에디터의 연동을 통한 사업계획서 자동 초안 생성.
    * **Control Tower:** 복잡한 지원 프로세스를 한눈에 파악하고 통제하는 고밀도 대시보드.

## 2. Tech Stack
* **Frontend:** React (Vite) + TypeScript
* **Styling:** Tailwind CSS (Dark Mode: Slate-950 Base)
* **Icons:** Material Symbols Outlined
* **State Management:** Zustand (Company Profile, Document Status, Editor State)
* **AI Integration:** OpenAI API (Document analysis & Drafting)

## 3. Core Modules
* **Intelligence Engine:** 기업 자격 요건 자동 필터링 및 공고별 성공 점수(Success Score) 산출.
* **Execution Engine (Vault & Editor):** 기업 공용 서류 관리 및 AI 기반의 사업계획서 자동 완성 에디터.
* **Status Monitoring:** 3-Column 레이아웃을 통한 실시간 서류 준비 현황(Checklist) 상시 노출.

## 4. Design Identity
* **Concept:** High-Tech Cockpit (관제실)
* **Visuals:** Deep Navy/Slate, Bold Typography (font-black), Neon Accent (Primary Blue/Action Orange).
