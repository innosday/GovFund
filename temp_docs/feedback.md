# GovFund: Strategic Funding Command Center - 중간 점검 보고서

## 1. 서비스 지향성 (Identity & Vision)
*   **슬로건:** "단순한 검색을 넘어, 자금 쟁취를 위한 전략 사령부"
*   **정체성:** 스타트업 및 중소기업이 복잡한 정부 지원사업(R&D)을 관제탑처럼 한곳에서 통제하고, AI의 보좌를 받아 합격 확률을 극대화하는 **B2B 전략 실행 플랫폼**.
*   **차별점:** 타 서비스가 '정보 나열'에 그칠 때, GovFund는 기업 데이터를 기반으로 **'성공 확률'**을 계산하고 **'서류 작성'**까지 자동화하는 실행 중심의 정체성을 가짐.

---

## 2. 핵심 기능 설명 및 위치

### ① 전략적 건강도 대시보드 (Strategic Dashboard)
*   **위치:** 사이드바 > '대시보드' 메뉴
*   **기능:** 기업의 현재 업력, R&D 투자 비중, 특허 보유 수 등을 기반으로 한 **'전략적 건강 점수'** 시각화 및 향후 자금 조달 로드맵 제공.
*   **필요성:** 우리 회사가 현재 지원금을 받기에 얼마나 준비되었는지(Ready)를 데이터로 직관적으로 인지시키기 위함.

### ② AI 스마트 매칭 엔진 (AI Intelligence Matcher)
*   **위치:** 사이드바 > 'AI 스마트 매칭' 메뉴
*   **기능:** 수천 개의 공고 중 우리 기업 정보와 일치하는 사업을 필터링하고, **'합격 확률(Success Score)'**을 % 수치로 제시.
*   **필요성:** 자격이 안 되는 사업에 낭비되는 행정력을 제거하고, 당첨 가능성이 가장 높은 사업에 집중(Focus)하게 함.

### ③ AI 전략 수립 에디터 (Execution Editor)
*   **위치:** 사이드바 > '전략 수립 에디터' (또는 매칭에서 '전략 수립 시작' 클릭 시)
*   **기능:** 
    *   **Vault 연동:** 클릭 한 번으로 기업의 기본 정보를 사업계획서 도입부에 자동 주입.
    *   **AI 실시간 피드백:** Gemini API를 통해 작성 중인 내용의 강점(Success), 약점(Warning), 가점 팁(Info)을 실시간으로 분석.
*   **필요성:** 가장 고통스러운 단계인 '사업계획서 초안 작성' 시간을 80% 이상 단축하고, 전문가 수준의 논리를 확보하기 위함.

### ④ 실시간 서류 관제 패널 (Status Panel)
*   **위치:** 화면 우측 고정 패널
*   **기능:** 20여 종의 필수 서류 구비 현황을 실시간 체크리스트로 표시. 클릭 시 가상 업로드 및 상태 업데이트.
*   **필요성:** 제출 직전 서류 누락으로 인한 탈락(미달)을 원천 차단하고, 전체 준비 프로세스를 가시화함.

### ⑤ 기업 핵심 데이터 관리 (Settings & Persistence)
*   **위치:** 사이드바 > '계정 설정' 메뉴
*   **기능:** 매출, 인원, 특허, 인증 여부 등 매칭의 기준이 되는 데이터 수정 및 **Firebase Firestore 클라우드 동기화**.
*   **필요성:** 사용자가 입력한 데이터가 영구적으로 보존되어 모든 분석 로직(매칭 점수 등)의 근거(Source of Truth)가 됨.

## 3. 기술적 완성도
*   **Frontend:** React + Tailwind CSS (고밀도 하이테크 디자인, Slate-950 테마)
*   **Backend:** Firebase Auth(인증) + Firestore(실시간 DB 데이터 영속성 확보)
*   **AI:** Google Gemini 1.5/3 Flash API (사업계획서 자동 생성 및 전략 분석)
*   **State:** Zustand (전역 상태 관리 및 낙관적 UI 업데이트 구현)
